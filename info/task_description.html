Input: A string.

Output: {% if interpreter.slug == "js-node" %}An array of strings{% else %}A list of strings{% endif %}

Example:

{% if interpreter.slug == "js-node" %} 
splitPairs('abcd') == ['ab', 'cd']
splitPairs('abc') == ['ab', 'c_']
{% else %} 
split_pairs('abcd') == ['ab', 'cd']
split_pairs('abc') == ['ab', 'c_']
{% endif %}
